# 2D Particle Simulator (rsparticle)

Rustで実装された2D粒子シミュレーター。異なる種類の粒子間の非対称な相互作用をシミュレートします。

## 特徴

- **非対称相互作用**: 粒子種類間の引力・斥力が非対称（AがBから受ける力とBがAから受ける力が異なる）
- **高精度物理演算**: Velocity Verlet積分法を使用した安定した物理シミュレーション
- **リアルタイムレンダリング**: macroquadを使用したスムーズな60FPSレンダリング
- **設定可能な相互作用**: N×N相互作用行列による柔軟な力の定義

## 実行方法

```bash
cargo run --release
```

## 操作方法

- **ESC**: プログラム終了

## シミュレーション設定

現在のデモでは3種類の粒子を使用：

- **赤い粒子（A）**: 緑い粒子に強く引かれる、青い粒子に弱く引かれる
- **緑い粒子（B）**: 赤い粒子から逃げる（斥力）、青い粒子と弱い引力
- **青い粒子（C）**: 青い粒子同士で反発、赤い粒子と緑い粒子に弱く引かれる

## 技術仕様

### 物理演算
- **積分法**: Velocity Verlet法
- **相互作用**: 万有引力型の力（距離の二乗に反比例）
- **境界処理**: ラップアラウンド（トーラス状空間）

### データ構造
- **Vec2**: glam::DVec2をベースとした2Dベクトル
- **ParticleType**: 粒子種類の定義（色、質量）
- **InteractionMatrix**: N×N相互作用定数行列
- **Simulator**: シミュレーション管理

### 計算量
- 力計算: O(N²) - 全粒子ペアの相互作用を計算

## ライブラリ

- **macroquad**: グラフィックスとゲームループ
- **glam**: 数学計算（ベクトル演算）
